# Services

## Rapid Landscape (1–2 weeks)
- Topic landscape: key themes, seminal papers, and gaps
- Curated reading list + annotated bibliography
- 1–2 summary figures (e.g., trend map, topic structure)
- Reference library (BibTeX/RIS)

## Structured Review Build (3–6 weeks)
- Agreed search strategy and screening plan
- Evidence table(s) + synthesis narrative
- Draft outline + section drafts (clearly marked as draft)
- Figures/tables + reference library + evidence trail appendix

## Revision & Evidence Upgrade (1–3 weeks)
- Strengthen an existing draft (structure, missing citations, argument flow)
- Add targeted evidence and 1–2 new visuals/tables
- Revision memo + tracked changes

Access is consultation-gated. **[Request a discussion](/request/)**
